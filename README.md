# CollabWhiteboard

A collaborative whiteboard application powered by React, Vite, and Google Gemini.

## Quick Start (Local)
1. Install [Node.js](https://nodejs.org/).
2. Open terminal in this folder.
3. Run `npm install`.
4. Open `.env` file and set your `VITE_GEMINI_API_KEY`.
5. Run `npm run dev`.

## Deployment
See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions on how to host this for free on Netlify (using GitHub or Drag-and-Drop).

## Features
- **Whiteboard**: Drag and drop notes, todos, calendar items.
- **AI Assistant**: Click the chat icon to talk to Gemini (supports Voice).
- **Persistence**: Your work is saved automatically.
